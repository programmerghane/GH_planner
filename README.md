# GH Planner v3 — AI Life Operating System

A production-ready AI-powered Life OS built with React, Vite, Express and Groq AI.

## Features
- 🤖 GH AI Coach powered by Groq (LLaMA 3.3)
- 📅 Smart Daily Planner with AI scheduling
- 🎯 Goals & Habits tracking
- 📊 Custom Trackers with analytics
- 📝 Notes with AI assistance
- 📆 Persian/Gregorian/Islamic Calendar
- 🏆 XP & Gamification system
- 🌙 Premium themes

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env`
3. Add your `GROQ_API_KEY` from [console.groq.com](https://console.groq.com)
4. Run `npm install && npm run dev`

## Deploy to Railway

Set these environment variables:
- `GROQ_API_KEY` — your Groq API key
- `PORT` — set to 8080 (Railway default)

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Express 4
- Groq AI (LLaMA 3.3 70B)
- Tailwind CSS v4
- Recharts
- jalaali-js (Persian calendar)
